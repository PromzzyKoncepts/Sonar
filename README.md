## Manual Installation

Clone the repo:

```bash
git clone https://github.com/PromzzyKoncepts/Sonar.git
cd Sonar
```

Install the dependencies:

```bash
npm install
```

Running in development:

```bash
npm run dev
```

### Deployment
```bash
npm run build
```
